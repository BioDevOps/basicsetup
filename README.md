# basicsetup
basic machine setup cookbook

# Prerequisite for vagrant
```
vagrant plugin install vagrant-chef-zero
vagrant plugin install vagrant-omnibus
```

# vagrant
```
berks vendor cookbooks
vagrant up
vagrant provision
```
